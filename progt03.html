Números aleatorios

Se utilizan para emular el comportamiento de algún fenómeno natural, el resultado de un juego de azar o para generar cualquier valor impredecible a priori.

Números aleatorios

Math.random() genera un double aleatorio tal que
0 <= número < 1

						 
							public static void main(String[] args) {
								// Declaración e inicialización de 3 números 
								// aleatorios entre 0 y 0.9999999999999999
								double aleatorio1 = Math.random();
								double aleatorio2 = Math.random();
								double aleatorio3 = Math.random();
						
								// En cada ejecución variará el valor devuelto
								System.out.println(aleatorio1); // 0.7885120757570676
								System.out.println(aleatorio2); // 0.7591427401039763
								System.out.println(aleatorio3); // 0.5081770986579679
							}

Números aleatorios

Rangos personalizados: Multiplicamos el aleatorio por el número de elementos del rango y le sumamos el Vi (Valor inicial del rango) convirtiendo todo a entero.

(int) (aleatorio * nº elementos + Vi)

						 
							// Ejemplo: Cara dado aleatoria
							public static void main(String[] args) {
								// Aleatorio entre 0 y 0.999999... con decimales
								double aleatorio = Math.random();
								// Convertimos a rango 1-6 caras posibles
								int caraDado = (int) (aleatorio * 6 + 1);
								System.out.println(caraDado); //
							}

Números aleatorios

Combinados con un switch se pueden generar palabras aleatorias

						 
                            public static void main(String[] args) {
                                System.out.println("Genera al azar piedra, papel o tijera:"); 
                                // genera un número al azar entre 0 y 2 (ambos incluidos)
                                int mano = (int) (Math.random() * 3); 

                                switch (mano) {
                                    case 0: System.out.println("piedra"); break;
                                    case 1: System.out.println("papel"); break;
                                    case 2: System.out.println("tijera"); break;
                                    default: 
                                }
                            }

Introducción

Sobrecarga de funciones

Java permite que varias funciones compartan el mismo identificador siempre que sus parámetros sean diferentes en número o tipo.

						 
							public static int suma(int a, int b){
								return a+b;
							}
						
							public static int suma(int a, int b, int c){
								return a+b+c;
							}
						
							public static float suma(float a, float b){
								return a+b;
							}

Ámbito de las variables

Una variable actúa dentro de su ámbito o bloque donde está definida.

Dos variables con el mismo nombre, definidas dentro y fuera de la función serán dos variables distintas.

Ámbito de las variables

						 
                            public static void main(String[] args) {
                                Scanner s = new Scanner(System.in);
								
                                System.out.print("Bienvenido, introduzca un número: ");
                                int i = Integer.parseInt(s.nextLine());
                                if (esPrimo(i)) System.out.println("El número 1 es primo"); 
                                else System.out.println("El número 1 no es primo"); 
                                
                                System.out.print("Introduzca otro número: ");
                                int x = Integer.parseInt(s.nextLine());
                                if (esPrimo(x)) System.out.println("El número 2 es primo"); 
                                else System.out.println("El número 2 no es primo");
                            }

                            public static boolean esPrimo(int x) { 
                                for (int i = 2; i < x; i++) {
                                    if ((x % i) == 0) {
                                        return false;
                                    } 
                                }
                                return true;
                            }

Parámetros por valor o referencia

Valor: Se pasa una copia de la variable, sólo importa su valor. Cualquier modificación que se haga a la variable dentro de la función no tendrá ningún efecto fuera.

Referencia: Si se modifica el valor de la variable dentro de la función, los cambios se mantienen fuera de la misma.

Parámetros por valor o referencia

Parámetros por valor o referencia

En muchos lenguajes es el programador quien decide cuándo se pasa un parámetro por valor o por referencia. En Java no podemos elegir.

Todos los parámetros que son de tipo primitivo (int, double, float, char, String...) se pasan siempre por valor mientras que arrays y objetos se pasan siempre por referencia.

Bibliotecas de funciones

Las funciones de un determinado tipo se pueden agrupar para crear un paquete que luego se importará desde el programa que necesite usarlas.

Cada paquete equivale a un directorio, por lo que podremos crear agrupaciones complejas mediante subpaquetes simplemente creando subdirectorios.

Bibliotecas de funciones

Declaración de las funciones en una clase dentro de un paquete.

						 
							package bibliotecaFunciones;

							public class Matematicas {
							
								public static int suma(int a, int b){
									return a+b;
								}
							
								public static int resta(int a, int b){
									return a-b;
								}
							}

Bibliotecas de funciones

Uso de las funciones

						 
							package funciones;

							public class Funciones01 {
								public static void main(String[] args) {
									int resultado = bibliotecaFunciones.Matematicas.suma(5, 6);
									System.out.println(resultado);
								}
							}

Bibliotecas de funciones

Uso de las funciones

						 
							package funciones;

							import static bibliotecaFunciones.Matematicas.suma;
							// import static bibliotecaFunciones.Matematicas.*;

							public class Funciones01 {

Recursividad

La recursividad es un concepto donde una función se llama a sí misma para resolver un problema. Este enfoque resulta especialmente útil para problemas que se pueden dividir en subproblemas más pequeños de la misma naturaleza.

Recursividad

¿Cómo funciona la recursividad?

  1. Caso base: Condición que detiene la recursión. Sin un caso base, la función se llamaría a sí misma infinitamente.
  2. Caso recursivo: Es la parte donde la función se llama a sí misma con una versión más simple del problema original.

Debemos garantizar que cada llamada se acerque al caso base.

Recursividad

Ejemplo práctico: Factorial

El factorial de un número n (representado como n!) es el producto de todos los números enteros positivos desde 1 hata n. Matemáticamente:

El factorial también se puede definir recursivamente:

Recursividad

Ejemplo práctico: Factorial

						 
							public class Recursividad {
								// Método recursivo para calcular el factorial
								public static int factorial(int n) {
									if (n == 0) { // Caso base
										return 1;
									} else { // Caso recursivo
										return n * factorial(n - 1);
									}
								}
							
								public static void main(String[] args) {
									int numero = 5;
									System.out.println("El factorial de " + numero + " es: " + factorial(numero));
								}
							}

Ejemplo de comentarios Javadoc

						 
                            /**
                            * Comprueba si un número entero positivo es primo o no.
                            * Un número es primo cuando únicamente es divisible entre * él mismo y la unidad.
                            *
                            * @author Thomas A. Anderson
                            * @param x un número entero positivo
                            * @return true si el número es primo
                            * @return false en caso contrario
                            */

                            public static boolean esPrimo(int x) { 
                                for (int i = 2; i < x; i++) {
                                    if ((x % i) == 0) {
                                        return false;
                                    } 
                                }
                                return true;
                            }
